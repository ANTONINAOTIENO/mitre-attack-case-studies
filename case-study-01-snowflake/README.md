# MITRE ATT&CK Case Studies

This repository contains my analysis of real-world cybersecurity incidents using the MITRE ATT&CK framework.

The goal of these case studies is to better understand attacker behavior by mapping publicly reported incidents to ATT&CK tactics and techniques, identifying detection opportunities, and exploring defensive mitigations.

Rather than focusing only on what happened, I focus on why attackers performed specific actions and how defenders could have detected or prevented them.

## Case Studies

### Case Study 01: Snowflake Customer Data Breaches (2024)

In this case study, I analyze the Snowflake customer data breaches that affected multiple organizations in 2024.

The analysis covers:

* Initial access through valid accounts
* Discovery of valuable data
* Collection of targeted information
* Exfiltration of sensitive data
* Detection opportunities
* Security mitigations
